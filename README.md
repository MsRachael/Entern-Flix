## Run Locally

Clone the project

```bash
  git clone https://github.com/MsRachael/Entern-Flix.git
```

Go to the project directory

```bash
  cd acharya
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
